CSS Development

1. CSS Selectors

CSS selectors define which HTML elements your styles will apply to. They are the foundation of CSS design.

  • Universal Selector: * { margin: 0; padding: 0; }
  • Type Selector: p { color: blue; }
  • Class Selector: .box { background: yellow; }
  • ID Selector: #main { border: 2px solid red; }
  • Attribute Selector: input[type="text"] { border-radius: 5px; }

Selectors can be combined for complex targeting. For example, div p applies styles to paragraphs inside divs only.

2. CSS Colors

CSS allows you to apply colors using names, HEX, RGB, RGBA, or HSL values.

  • Named Color: color: red;
  • HEX Value: color: #ff0000;
  • RGB: color: rgb(255,0,0);
  • RGBA (with transparency): color: rgba(255,0,0,0.5);
  • HSL: color: hsl(0, 100%, 50%);

Colors are not limited to text; you can color borders, backgrounds, and shadows as well.

3. CSS Backgrounds

Backgrounds enhance the visual appeal of web pages by applying colors, gradients, or images.

Examples:

  • background-color: lightblue;
  • background-image: url('bg.jpg');
  • background-repeat: no-repeat;
  • background-size: cover;
  • background-position: center;

CSS gradients allow smooth transitions between colors using linear-gradient() or radial-gradient().

4. CSS Borders

Borders help visually separate or highlight elements.

Examples:

  • border: 2px solid black;
  • border-radius: 10px;
  • border-style: dotted dashed solid double;

Rounded borders give a modern, soft look to containers.

5. CSS Box Model

The box model explains how every element is a rectangular box with four layers: content, padding, border, and margin.

div { width: 200px; padding: 20px; border: 5px solid black; margin: 15px; }

Understanding the box model helps fix layout issues and design responsive pages.

6. CSS Shadows

Shadows add depth and realism to designs. CSS supports both text and box shadows.

Examples:

  • Box Shadow: box-shadow: 4px 4px 15px rgba(0,0,0,0.3);
  • Text Shadow: text-shadow: 2px 2px 5px gray;

Shadows can create soft glowing effects, 3D designs, or floating element visuals.
